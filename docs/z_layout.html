grid

  • レイアウトを均等に分けてマス目上に考えるレイアウトをする際に使用します。
  • 固定カラムはtheme内で定義を行ってください。
  • 固定カラムを使用する場合は予めサイズを計算して、それぞれの値を付与したクラスを使用してください。
  • レスポンシブ対応する場合は、固定カラムは非推奨です。レイアウトが崩れます。
  • 固定値と相対値を混同しないようにしてください。
  • モジュール内のレイアウトには使用しません。別のクラスを利用してください。

table_grid

  • グリッドでカラムレイアウトをし、更にインライン要素の縦方向を操作したい際に使用します。
  • 固定カラムはtheme内で定義を行ってください。
  • 固定カラムを使用する場合は予めサイズを計算して、それぞれの値を付与したクラスを使用してください。
  • レスポンシブ対応する場合は、固定カラムは非推奨です。レイアウトが崩れます。
  • 固定値と相対値を混同しないようにしてください。
  • 主にモジュール内のレイアウトに使用します。
